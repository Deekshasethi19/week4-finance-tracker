# week4-finance-tracker
A Python-based Personal Finance Tracker that helps users manage daily expenses efficiently. It supports expense management, JSON and CSV file handling, budget tracking, reports, search and filter options, recurring expenses, and backup &amp; restore functionality using object-oriented programming.

## Overview

Personal Finance Tracker is a Python application that helps users manage daily expenses. It supports expense management, reports, budget tracking, JSON/CSV file handling, search, filter, backup, and restore features.

## Features

- Add, Update & Delete Expenses
- Search & Filter Expenses
- Budget Tracking
- Category & Monthly Reports
- JSON & CSV File Handling
- Backup & Restore
- Recurring Expenses

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- JSON
- CSV

## How to Run

```bash
python main.py
```
